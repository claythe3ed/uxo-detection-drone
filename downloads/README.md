Sources of PDF Files Used in This Project

This directory contains the original scientific papers and reports used to design and simulate the UXO Detection Drone system. These files are not uploaded to the GitHub repository due to their size and copyright considerations. They can be obtained from their original sources as listed below.

1. GICHD / ICRC Report
Title: The Use of Remote Sensing and Artificial Intelligence in the Mine Action Sector
Publisher: Geneva International Centre for Humanitarian Demining (GICHD)
Link: https://www.gichd.org/fileadmin/uploads/gichd/Publications/ICRC_GICHD_Webinar_Report_-_The_Use_of_Remote_Sensing_and_Artificial_Intelligence_in_the_Mine_Action_Sector.pdf

2. Maggy Platform Paper
Title: Intelligent, Automated, Rapid, and Safe Landmine, Improvised Explosive Device and Unexploded Ordnance Detection Using Maggy
Authors: Kuru et al.
Published in: IEEE Access, 2024
DOI: 10.1109/ACCESS.2024.3493013
Open-access author copy: https://clok.uclan.ac.uk/id/eprint/49066/8/49066%20Kuru%20et%20al.%20VOR.pdf

3. Drone Magnetometer Interference Removal Paper
Title: A Drone-mounted Magnetometer System for UXO Detection: Active Interference Cancellation and Field Tests
Source: arXiv, reference number 2510.01417
Link: https://arxiv.org/pdf/2510.01417

No additional PDFs were used. All other data files in this repository were generated locally or obtained via open APIs such as Open-Meteo, SoilGrids, and OpenTopoData. These three papers form the scientific foundation of the simulation models and the drone design. Anyone wishing to replicate or extend this work should begin with these sources.
